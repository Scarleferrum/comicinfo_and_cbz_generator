## A Basic Tool to Creating CBZ E-books
A basic Tool to creating CBZ e-books

## Instructions:
This script depends on the following modules: python, pandas, openpyxl, and tkinterdnd2.  
Basic programming knowledge is required.

1. Install python  
- You can download it from the official python website. Make sure to check the option to install pip during setup.

2. Install dependencies  
- Run the following command in your terminal  
Please copy and execute the following code in the terminal  
```pip install pandas openpyxl tkinterdnd2```  
If you encounter errors, try:  
```pip3 install pandas openpyxl tkinterdnd2```

3. Run comicinfo_gui_v2.4.3.py

4. Choose different modes based on different data structures for compression

### Mode Descriptions:
1. Mode 1: Suitable for fully organized archives or folders  
- Supports drag-and-drop for .zip files or folders  
- Supports browsing and selecting .zip files or folders  
- Supported image formats: .jpeg/.jpg, .png, .webp  
- Image files must be properly ordered  
    Correct format:  
        001, 002, 003, 004, 005...099  
    Incorrect format:  
        1, 2, 3, 4, 5...99  
- Title must be filled in  

2. Mode 2: Suitable for comics purchased from DLsite; different circles may have different data structure  
- Requires completing the Excel file first  
- Folder structure and file organization must be arranged manually  
- Batch generates .xml and .cbz files
